# Google Play Store Apps Exploratory Data Analysis (EDA)

## Overview
This project focuses on performing Exploratory Data Analysis (EDA) on the Google Play Store dataset to extract meaningful insights about app performance, user engagement, and market trends.

The dataset contains information about thousands of Android applications, including their category, ratings, reviews, size, installs, and pricing.

---

## Objectives
- Analyze app distribution across different categories
- Understand factors affecting app ratings and installs
- Identify trends in free vs paid apps
- Extract actionable insights from real-world data

---

## Tools & Technologies
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn

---

## Dataset

The dataset used in this project is the Google Play Store Apps dataset from Kaggle, containing over 10,000 apps.

---

## Future Improvements
- Build interactive dashboards (Power BI/Tableau)

## Conclusion

This analysis demonstrates how raw app data can be transformed into meaningful insights.
Understanding these trends can help businesses and developers make better decisions regarding app development, pricing, and marketing strategies.
